## 📊 评分系统详解

### 技术评分（0-100）

| 评分区间 | 含义 | 投资评级 |
|---------|------|---------|
| 80-100 | 技术形态优秀 | A+ |
| 70-79 | 技术形态良好 | A |
| 60-69 | 技术形态尚可 | B+ |

```
技术评分 = 趋势评分×40% + 量价评分×30% + 动量评分×30%

## 🚨 重要风控原则

### 1. 止损纪律（最重要！）
```
✅ 严格执行止损，不抱幻想
✅ 止损价一旦触发，立即平仓
✅ 不要因为"账面浮亏"而不愿止损
✅ 记住：止损是保护本金，不是认输
```

### 2. 仓位管理
```
✅ 单只股票不超过40%
✅ 高风险标的不超过10%
✅ 总仓位根据市场环境调整：
   - 进攻环境：70-80%
   - 中性环境：40-50%
   - 防守环境：20-30%
```

### 3. 分批建仓
```
✅ 第一批：50%仓位（试探性建仓）
✅ 第二批：30%仓位（回调或突破后加仓）
✅ 第三批：20%仓位（趋势确认后加仓）

❌ 不要一次性全仓买入
❌ 不要追涨杀跌
```

### 4. 回撤管理
```
✅ 设定最大回撤容忍度（如15%）
✅ 触及回撤线后暂停交易
✅ 等待市场环境改善
✅ 重新评估策略
```

### 技巧3：风险收益比筛选
```

❌ 风险收益比 < 1.5：不佳，暂时观望
```

```

```

```

### Q2：个股评级是"推荐"，但市场前瞻是"防守"，怎么办？

```

✅ 市场防守 → 整体降低风险敞口
✅ 即使个股优秀，也不宜重仓
✅ 仓位减半（推荐15-25% → 实际8-12%）
✅ 等待市场环境改善
```
