核心功能说明
═══════════════════════════════════════════════════════════════

1. auto_reporter.py
   - 获取市场数据（美股/港股/A股）
   - 调用 Gemini AI 生成分析报告
   - 推送到钉钉群
   - 支持早报/晚报两种模式

2. GitHub Actions (daily-report.yml)
   - 每天 7:00 自动运行（早报）
   - 每天 19:00 自动运行（晚报）
   - 从 Secrets 读取密钥
   - 完全自动化，无需人工干预

3. 安全机制
   - 敏感信息存储在 GitHub Secrets
   - 钉钉加签验证
   - .gitignore 防止密钥泄露
