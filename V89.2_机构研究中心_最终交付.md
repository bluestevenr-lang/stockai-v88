### 1. **专业机构级分析框架**
采用机构投研标准流程：
```
宏观市场前瞻
    ↓
个股深度研究（技术+量价+趋势）
    ↓
机会雷达（量化评分+催化剂）
    ↓
风险预警（多维度评估+止损管理）
    ↓
综合投资建议（评级+操作+仓位）
```

### 2. **前瞻性分析能力**
- ✅ 趋势外推（基于MA系统和技术指标）

- ✅ 时间视野管理（1-2周/1-2月/3-6月）

### 3. **机会风险评估**
- ✅ 机会评分（0-100分，技术+市场+风险收益比）
- ✅ 风险评分（0-100分，多维度风险因素）
- ✅ 入场时机建议（立即/等待回调/观望）
- ✅ 仓位管理建议（重仓/中等/轻仓/空仓）

#### 1. 市场前瞻分析：`analyze_market_outlook()`

**功能**：基于全球市场体制+技术分析，预判未来市场走向

**输入**：
- `market_regime`: 市场体制数据（Risk On/Off/Neutral）

**输出**：
```python
{
    'trend_direction': '强势上升趋势/上升趋势/下降趋势/震荡整理',
    'strength_score': 0-100,  # 趋势强度评分
    'key_levels': {          # 关键技术位
        'resistance_1': 460.50,  # 第一阻力位
        'resistance_2': 465.20,  # 第二阻力位
        'support_1': 455.12,     # 第一支撑位（MA50）
        'support_2': 448.30      # 第二支撑位（MA200）
    },
    'catalyst': [            # 未来催化剂
        '低波动环境，趋势延续概率高',
        '风险偏好回升，资金流入风险资产'
    ],
    'time_horizon': '1-2月',
    'confidence': 80,        # 置信度
    'recommendation': '进攻/防守/观望'
}
```

**分析逻辑**：

| 趋势判断 | 条件 | 强度评分 |
|---------|------|----------|
| 强势上升趋势 | SPY > MA50 > MA200 | 85 |
| 上升趋势 | SPY > MA50 | 65 |
| 弱势下降趋势 | SPY < MA50 < MA200 | 35 |
| 下降趋势 | SPY < MA200 | 15 |
| 震荡整理 | 其他 | 50 |

**操作建议规则**：
- **进攻**：Risk On + 低VIX（<15）
- **防守**：Risk Off + 高VIX（>25）
- **观望**：其他情况

#### 2. 个股深度研究：`deep_research_stock()`

**功能**：技术+量价+趋势+动量的全方位分析

**输入**：
- `stock_data`: 股票历史数据（OHLCV）
- `code`: 股票代码

**输出**：
```python
{
    'technical_score': 75,  # 综合技术评分（0-100）
    'trend_analysis': {      # 趋势分析
        'status': '多头趋势',
        'score': 70,
        'ma5': 458.50,
        'ma20': 455.20,
        'ma50': 450.10,
        'deviation_from_ma20': +1.45  # 偏离MA20百分比
    },
    'volume_analysis': {     # 量价分析
        'status': '温和放量',
        'score': 70,
        'volume_ratio': 1.65,  # 成交量比率
        'avg_volume_20d': 5234500
    },
    'momentum_signals': [    # 动量信号
        '📈 MACD金叉，多头信号',
        '📊 RSI中性区域，趋势待确认'
    ],
    'price_target': {        # 价格目标
        'current_price': 458.32,
        'target_high': 472.50,   # 上涨目标
        'target_low': 445.80,    # 下跌目标
        'stop_loss': 442.00,     # 止损价
        'time_horizon': '1-2月'
    },
    'risk_reward_ratio': 2.35,  # 风险收益比
    'investment_rating': 'A'    # 投资评级（A+/A/B+/B/C）
}
```

**评分体系**：

**1. 趋势评分（40%权重）**

| 趋势状态 | 条件 | 评分 |
|---------|------|------|
| 强势多头排列 | 价格 > MA5 > MA20 > MA50 | 90 |
| 多头趋势 | 价格 > MA5 > MA20 | 70 |
| 震荡整理 | 其他 | 50 |
| 空头趋势 | 价格 < MA5 < MA20 | 40 |
| 弱势空头排列 | 价格 < MA5 < MA20 < MA50 | 20 |

**2. 量价评分（30%权重）**

| 量能状态 | 条件 | 评分 |
|---------|------|------|
| 放量突破 | 成交量比 > 2.0 | 85 |
| 温和放量 | 成交量比 > 1.5 | 70 |
| 量能正常 | 0.5 < 成交量比 < 1.5 | 60 |
| 缩量整理 | 成交量比 < 0.5 | 40 |

**3. 动量评分（30%权重）**

| 信号类型 | 条件 | 加分 |
|---------|------|------|
| MACD金叉（零轴上） | MACD > Signal & MACD > 0 | +25 |
| MACD金叉 | MACD > Signal | +20 |
| RSI超卖 | RSI < 30 | +20 |
| RSI中性 | 40 < RSI < 60 | +10 |

**投资评级标准**：
- **A+**: 技术评分 ≥ 80
- **A**: 技术评分 ≥ 70
- **B+**: 技术评分 ≥ 60
- **B**: 技术评分 ≥ 50
- **C**: 技术评分 < 50

#### 3. 机会雷达：`opportunity_radar()`

**功能**：综合评分系统，识别高机会标的

**输入**：
- `stock_analysis`: 个股深度研究结果
- `market_outlook`: 市场前瞻分析结果

**输出**：
```python
{
    'opportunity_score': 78,  # 综合机会评分（0-100）
    'opportunity_level': '🔥 高机会',
    'entry_timing': '立即布局',
    'catalysts': [
        '技术形态良好，多头排列',
        '📈 MACD金叉，多头信号',
        '风险偏好回升，资金流入风险资产'
    ],
    'optimal_entry_price': 458.32,
    'position_size_suggestion': '重仓（30-40%）'
}
```

**机会评分公式**：
```
机会评分 = 技术评分 × 40% + 市场强度 × 30% + 风险收益比评分 × 30%
```

**机会等级划分**：

| 机会等级 | 评分范围 | 入场时机 | 建议仓位 |
|---------|---------|---------|---------|
| 🔥 高机会 | ≥ 75 | 立即布局 | 重仓（30-40%） |
| 📈 中等机会 | 60-74 | 适度参与 | 中等（15-25%） |
| 📊 观察机会 | 45-59 | 等待回调 | 轻仓（5-10%） |
| ⚠️ 机会不足 | < 45 | 暂时观望 | 空仓等待 |

**催化剂分析逻辑**：
1. 技术面催化剂：多头排列、金叉信号、超卖反弹
2. 市场环境催化剂：风险偏好回升、资金流入
3. 动量催化剂：成交量放大、突破关键位

**最优入场价计算**：
- 机会评分 ≥ 70：接受当前价
- 机会评分 < 70：等待回调至MA20附近（MA20 × 1.02）

#### 4. 风险预警：`risk_warning()`

**功能**：多维度风险评估+缓释建议

**输出**：
```python
{
    'risk_level': '🟡 中等风险',
    'risk_score': 45,  # 风险评分（0-100）
    'risk_factors': [
        '⚠️ 技术形态弱势，趋势不明',
        '📊 风险收益比不佳（<1.5）'
    ],
    'stop_loss_price': 442.00,
    'max_drawdown_tolerance': 0.15,  # 最大回撤容忍（15%）
    'risk_mitigation': [
        '控制仓位在20%以内',
        '分批建仓，降低风险'
    ]
}
```

**风险评分因子**：

| 风险因子 | 触发条件 | 风险加分 |
|---------|---------|---------|
| 技术弱势 | 技术评分 < 40 | +25 |
| 空头趋势 | 趋势包含"空头" | +30 |
| RSI超买 | RSI > 70 | +20 |
| MACD死叉 | MACD < Signal | +20 |
| 市场偏弱 | 市场强度 < 40 | +25 |
| 防守建议 | 市场建议="防守" | +20 |
| 风险收益比低 | RR比 < 1.5 | +15 |

**风险等级划分**：

| 风险等级 | 评分范围 | 最大回撤容忍 | 缓释建议 |
|---------|---------|-------------|---------|
| 🔴 高风险 | ≥ 70 | 8% | 降低仓位至10%以下，严格止损 |
| 🟡 中等风险 | 40-69 | 15% | 控制仓位在20%以内，分批建仓 |
| 🟢 低风险 | < 40 | 20% | 正常仓位配置，持续跟踪 |

### 场景1：开盘前 - 查看市场前瞻

### 场景2：选股后 - 个股深度评估
```

3. 查看"机构研究报告"

   - 风险预警：风险因素+止损价+缓释建议
4. 根据报告决定：买入/等待/回避
```

```
机会评分 = 78分（高机会）

```
