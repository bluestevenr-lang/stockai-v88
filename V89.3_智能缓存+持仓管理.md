## ✅ 功能1：智能缓存系统

### 设计方案

**选择方案2（更智能）**：
- ✅ **工作日（周一到周五）**：10分钟缓存
- ✅ **休息日（周六日）**：24小时缓存
- ✅ **交易时段（9:30-16:00 ET）**：5分钟缓存（更激进）

**为什么选择方案2**：
1. 工作日市场开盘，数据变化快，10分钟刷新合理
2. 休息日市场休市，数据不变，24小时缓存节省资源
3. 交易时段内，数据更新更频繁，5分钟缓存提供更实时的数据

#### 2. 智能缓存函数（+57行）

```python
def get_smart_cache_ttl(data_type: str = 'daily') -> int:
    """
    智能缓存TTL - 根据当前时间和市场状态返回合适的TTL
    
    规则：
    - 工作日（周一到周五）：10分钟
    - 休息日（周六日）：24小时
    - 交易时段（更激进）：5分钟
    
    参数：
        data_type: 数据类型（'fast'/'daily'/'weekly'）
    
    返回：
        TTL秒数
    """
    from datetime import datetime
    import pytz
    
    # 获取当前时间（美东时间）
    now_et = datetime.now(pytz.timezone('America/New_York'))
    weekday = now_et.weekday()  # 0=周一, 6=周日
    hour = now_et.hour
    
    # 判断是否为休息日（周六日）
    if weekday >= 5:  # 5=周六, 6=周日
        return Config.CACHE_TTL_WEEKEND  # 24小时
    
    # 工作日
    # 判断是否在交易时段（美股：9:30-16:00 ET）
    if 9 <= hour < 16:
        return Config.CACHE_TTL_TRADING_HOURS  # 5分钟
    else:
        return Config.CACHE_TTL_WORKDAY  # 10分钟
```

**时区说明**：
- 使用美东时间（America/New_York）
- 美股交易时段：9:30-16:00 ET
- 自动处理夏令时

### 后端模块：portfolio_manager.py（279行）

#### 核心类：`PortfolioManager`

**主要方法**：

1. **`create_template()`** - 创建Excel模板

3. **`calculate_portfolio_metrics()`** - 计算持仓指标
   - 输入：持仓DataFrame + 当前价格
   - 计算：持仓市值、盈亏金额、盈亏比例
   - 返回：包含完整指标的DataFrame

4. **`get_portfolio_summary()`** - 获取持仓汇总
   - 总市值
   - 总成本
   - 总盈亏
   - 总盈亏比例
   - 盈利/亏损股票数

5. **`add_stock()`** - 添加持仓股票
   - 追加新持仓到Excel
   - 自动保存

---

#### 计算逻辑

```python
# 持仓市值 = 持仓数量 × 当前价格
持仓市值 = 100 × 185.00 = 18,500.00

# 持仓成本 = 持仓数量 × 买入价格
持仓成本 = 100 × 150.00 = 15,000.00

# 盈亏金额 = 持仓市值 - 持仓成本
盈亏金额 = 18,500.00 - 15,000.00 = 3,500.00

# 盈亏比例 = 盈亏金额 / 持仓成本 × 100%
盈亏比例 = 3,500.00 / 15,000.00 × 100% = 23.33%
```
