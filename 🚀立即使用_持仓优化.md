## 💡 新功能说明

### 1. 价格缓存状态显示

在持仓面板顶部会看到：
```
💾 价格缓存: 有效 | 更新时间: 2026-02-08 19:30:45 | 剩余: 23.5小时
```

**3种状态**：
- ✅ **有效**：显示剩余时间，使用缓存
- ⏰ **已过期**：提示将重新获取
- 🆕 **无缓存**：首次获取

---

### 2. 强制刷新价格

如果觉得价格不够新，可以手动刷新：

**按钮位置**：价格区域下方
```
[🔄 强制刷新价格]  [⏰ 价格数据时间: 19:30:45]  [💡 价格每天自动更新一次]
```

**操作**：
1. 点击 "🔄 强制刷新价格"
2. 清除缓存并重新获取（2-3分钟）
3. 更新后保存到缓存
4. 后续24小时内继续使用缓存

## 🔍 常见问题

### Q1: 为什么还要等2-3分钟？
**A**: 只有首次获取价格时需要。获取18只股票的最新价格需要时间，这是正常的。但获取一次后，24小时内都不需要再等！

### Q2: 如何知道价格是否最新？
**A**: 查看缓存状态：
```
💾 价格缓存: 有效 | 更新时间: 2026-02-08 19:30:45 | 剩余: 23.5小时
```
如果不满意，点击"🔄 强制刷新价格"。

### Q3: 编辑后为什么要重新获取价格？
**A**: 因为你可能删除或添加了股票，原来的价格缓存不再适用。系统智能清除缓存，确保数据准确。

### Q4: 删除股票怎么操作？
**A**: 
1. 点击 "✏️ 在线编辑持仓"
2. 勾选左侧的复选框 ☑️
3. 按键盘 Delete 键
4. 点击 "💾 保存持仓"

### Q5: 可以一次删除多只股票吗？
**A**: 可以！勾选多个复选框，然后按 Delete 键，就可以批量删除。

### Q6: 缓存会占用很多空间吗？
**A**: 不会。只缓存18只股票的价格数据（不到1KB），完全可以忽略不计。
