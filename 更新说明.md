### 1. 【关键修复】ProxyContext 类集成 ✅
- **问题**：之前代理配置不稳定，美股/A股数据无法正常获取
- **解决方案**：集成 V72 的 `ProxyContext` 类，强制设置环境变量
- **端口**：默认 1082（用户可在侧边栏动态修改）
- **代码位置**：行 207-225

```python
class ProxyContext:
    """[核心工具] 临时环境变量注入，强迫 YFinance 走指定代理"""
    def __enter__(self):
        os.environ["HTTP_PROXY"] = self.proxy_url
        os.environ["HTTPS_PROXY"] = self.proxy_url
        os.environ["ALL_PROXY"] = self.proxy_url
```

### 2. 【新增】专业量化回测指标函数 ✅
- **函数名**：`calculate_advanced_quant(df)`
- **位置**：行 424-464
- **计算周期**：最近 252 个交易日（约 1 年）
- **输出指标**：
  1. **夏普比率 (Sharpe Ratio)**：风险调整后收益，假设无风险利率 3%
  2. **最大回撤 (Max Drawdown)**：历史最高点回撤幅度
  3. **年化波动率 (Volatility)**：价格波动标准差
  4. **胜率 (Win Rate)**：日收益 > 0 的天数占比
  5. **盈亏比 (Profit/Loss Ratio)**：平均盈利 / 平均亏损

**容错机制**：数据不足时显示"计算中"
