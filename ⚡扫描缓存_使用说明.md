## 🎯 核心功能

### 1. 自动缓存（24小时）

```
扫描一次 → 保存结果 → 24小时有效
```

**显示提示**：
```
📦 使用缓存扫描结果 | 扫描时间: 10:30:45 | 剩余有效期: 23.5小时
```

### 2. 状态清晰

**3种状态**：
- ✅ **有效缓存**：显示剩余时间，使用缓存
- ⏰ **已过期**：提示重新扫描
- 🆕 **无缓存**：首次扫描

### 3. 灵活控制

**清除缓存按钮**：
```
[🗑️ 清除扫描缓存]
```
点击后可立即重新扫描获取最新数据

## ❓ 常见问题

### Q1: 缓存会自动更新吗？
**A**: 不会。缓存24小时固定不变，除非：
- 超过24小时自动过期
- 手动清除缓存
- 重新扫描覆盖

### Q2: 多个扫描结果能同时保存吗？
**A**: 不能。只保存最后一次扫描的结果。如需保存多个榜单，请使用"📥 导出CSV"功能。

### Q3: 缓存占用空间大吗？
**A**: 很小。只保存扫描结果数据（通常<100KB），完全可以忽略。

### Q4: 关闭应用后缓存还在吗？
**A**: 在。缓存保存在浏览器session中，除非清除浏览器数据。
