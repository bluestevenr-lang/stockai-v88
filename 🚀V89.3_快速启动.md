## 🎯 核心功能

### 功能1：智能缓存

**自动根据时间调整缓存TTL**：
- 工作日交易时段（9-16点）：**5分钟**
- 工作日其他时段：**10分钟**
- 休息日（周六日）：**24小时**

**好处**：
- ✅ 交易时段数据更实时
- ✅ 休息日节省API调用
- ✅ 自动适配，无需手动配置

---

### 功能2：Excel持仓管理

**使用Excel管理持仓**：
1. 应用自动创建`my_portfolio.xlsx`模板
2. 用户手动编辑Excel添加持仓
3. 前端自动显示实时盈亏
4. 一键对持仓股票进行AI分析

**Excel格式**：
```
| 股票代码   | 股票名称  | 持仓数量 | 买入价格 | 买入日期   | 备注     |
|-----------|----------|---------|---------|-----------|----------|
| AAPL      | 苹果     | 100     | 150.00  | 2024-01-01| 长期持有 |
| 600519.SS | 贵州茅台  | 200     | 1680.00 | 2024-03-10| 核心资产 |
```

1. 刷新页面（Ctrl+R或F5）
2. 查看持仓汇总和明细

### 场景3：智能缓存自动优化

**周一上午10:00（交易时段）**：
- 缓存TTL：5分钟
- 数据更新：每5分钟刷新
- 适用场景：盘中实时监控

**周一晚上20:00（盘后）**：
- 缓存TTL：10分钟
- 数据更新：每10分钟刷新
- 适用场景：盘后分析

**周六上午10:00（休息日）**：
- 缓存TTL：24小时
- 数据更新：全天不刷新
- 适用场景：周末复盘

## 🎯 最佳实践

### 1. 定期更新持仓

**建议频率**：
- 每次交易后立即更新
- 或每周更新一次

**操作步骤**：

2. 添加/删除/修改持仓
3. 保存文件
4. 刷新应用页面

---

### 2. 使用备注分类

**按投资策略分类**：
```
| 备注 |
|------|
| 长期持有 |
| 短期交易 |
| 价值投资 |
| 成长股   |
| 分红股   |
```

**按行业分类**：
```
| 备注 |
|------|
| 科技股 |
| 消费股 |
| 金融股 |
| 医药股 |
| 新能源 |
```

---

### 3. 结合AI分析调整持仓

**决策流程**：
1. 查看持仓盈亏
2. 选择股票进行AI分析
3. 查看机构研究报告：
   - 投资评级（A+/A/B+/B/C）
   - 机会评分（0-100）
   - 风险评分（0-100）
   - 操作建议（买入/观望/回避）
4. 根据报告决定：
   - 加仓（机会评分高，风险评分低）
   - 减仓（风险评分高）
   - 持有（评级良好）
   - 清仓（评级差，风险高）

---

### 4. 设置止损止盈

**在Excel中记录目标价**：
```
| 股票代码 | 买入价格 | 止损价 | 止盈价 | 备注 |
|---------|---------|-------|-------|------|
| AAPL    | 150.00  | 135.00| 180.00| 10%止损,20%止盈 |
```

**结合AI分析的止损价**：
- 风险预警中的"止损价"
- 技术分析中的"支撑位"

```

**解决方法**：

```

---

### Q3：Excel文件找不到

**检查事项**：
1. 文件名是否为`my_portfolio.xlsx`
2. 文件是否在StockAI目录下
3. 文件是否被Excel占用（关闭Excel再试）

---

### Q4：持仓数据显示"无数据"

**可能原因**：
1. 股票代码格式错误 → 检查格式（美股直接代码，A股加.SS/.SZ，港股加.HK）
2. 网络问题 → 检查网络连接
3. 代理设置 → 检查代理配置

---
