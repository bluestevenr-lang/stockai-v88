## ✅ ADD 列表（新增内容）

### 1. ExpectationLayer类扩展（+132行）

#### 新增方法1：analyze_hk_market_regime()
**功能**：分析港股市场体制（基于恒生指数）

**数据来源**：
- ^HSI：恒生指数

**分析指标**：
- MA50 / MA200
- 20日年化波动率
- 市场体制裁决

**波动率分级**（港股特色）：
| 波动率 | 状态 | 说明 |
|--------|------|------|
| > 35% | ⚠️ 高波动（谨慎） | 市场震荡剧烈 |
| 25-35% | 📈 中高波动（正常） | 港股常态 |
| < 15% | 📉 低波动（平稳） | 趋势稳定 |
| 15-25% | 📊 中等波动（均衡） | 震荡行情 |

**返回字段**：
```python
{
    'verdict': 'Risk On',
    'index_level': 26559.95,    # 恒指点位
    'volatility': 28.5,         # 年化波动率
    'vol_status': '📈 中高波动（正常）',
    'ma50': 26200.0,
    'ma200': 25800.0,
    'reason': '恒指(26560) > MA50(26200)；波动率=28.5%（温和）',
    'data_ok': True,
    'market_name': '港股'
}
```

---

#### 新增方法2：analyze_cn_market_regime()
**功能**：分析A股市场体制（基于上证指数）

**数据来源**：
- 000001.SS：上证指数

**分析指标**：
- MA50 / MA200
- 20日年化波动率
- 市场体制裁决

**波动率分级**（A股特色）：
| 波动率 | 状态 | 说明 |
|--------|------|------|
| > 40% | ⚠️ 高波动（政策敏感期） | 政策影响大 |
| 30-40% | 📈 中高波动（活跃） | 交易活跃 |
| < 20% | 📉 低波动（盘整） | 横盘整理 |
| 20-30% | 📊 中等波动（正常） | A股常态 |

**返回字段**：
```python
{
    'verdict': 'Neutral',
    'index_level': 3089.26,     # 上证点位
    'volatility': 25.8,         # 年化波动率
    'vol_status': '📊 中等波动（正常）',
    'ma50': 3100.0,
    'ma200': 3050.0,
    'reason': '上证在MA50(3100)与MA200(3050)之间',
    'data_ok': True,
    'market_name': 'A股'
}
```

---

#### 新增方法3：analyze_all_markets()
**功能**：并发分析三大市场 + 综合联动

**返回结构**：
```python
{
    'us_market': {...},   # 美股分析结果
    'hk_market': {...},   # 港股分析结果
    'cn_market': {...},   # A股分析结果
    'summary': {          # 综合分析
        'global_verdict': '🟢 全球风险偏好',
        'global_reason': '三大市场中2个处于Risk On状态',
        'risk_on_count': 2,
        'risk_off_count': 0,
        'valid_markets': 3
    }
}
```

**综合裁决规则**：
- Risk On市场 ≥ 2 → "🟢 全球风险偏好"
- Risk Off市场 ≥ 2 → "🔴 全球避险模式"
- 其他 → "🟡 市场分化（结构性行情）"
