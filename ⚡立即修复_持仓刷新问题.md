## 💡 最常见的问题和解决方法

### 问题1: Excel还在运行 ⚠️

**症状**: 
- 保存了文件
- 但修改时间没有更新

**解决**: 
```bash
macOS:
Cmd+Tab 切换到Excel → Cmd+Q 完全退出

Windows:
Ctrl+Shift+Esc 打开任务管理器 → 结束Excel进程
```

---

### 问题2: 编辑错文件 ⚠️

**症状**:
- 明明有数据
- 但应用显示0只

**解决**:
```bash
1. 查看应用中的"📁 文件位置"
2. 确认你编辑的是这个路径的文件
3. 使用"📂 打开Excel编辑"自动打开正确文件
```

---

### 问题3: 数据格式错误 ⚠️

**症状**:
- Excel有10只股票
- 但只显示4只

**解决**:
```bash
1. 点击"📋 查看Excel原始数据"
2. 对比Excel文件
3. 找出缺失的股票
4. 检查数据:
   - 持仓数量是数字吗？>0吗？
   - 买入价格是数字吗？>0吗？
5. 修正后保存
```
