## 🤖 进阶：通过钉钉消息更新持仓（可选）

**注意：这需要额外配置，适合高级用户**

如果你想通过钉钉发送消息来更新持仓股，需要：

### 步骤1：创建钉钉Outgoing机器人

1. 钉钉群设置 → 智能群助手 → 添加机器人
2. 选择「自定义（通过Webhook接入）」
3. 开启「消息接收模式」
4. 配置POST地址（需要一个公网可访问的服务器）

### 步骤2：部署Webhook接收器

这需要一个服务器来接收钉钉消息，比较复杂。

**推荐：不使用这个方案，直接用方式1-3更简单实用。**

## 💡 常见问题

### Q1: 修改Excel后需要重启GitHub Actions吗？

**A:** 不需要！只要把修改后的Excel推送到GitHub，下次定时运行（早7点/晚7点）会自动使用新的持仓股。

### Q2: 如何立即生成包含新持仓股的日报？

**A:** 
```bash
# 本地测试
python3 auto_reporter.py

# 或者在GitHub页面手动触发Actions
Actions → AI股市早晚报 → Run workflow
```

### Q3: 持仓股太多会增加成本吗？

**A:** 会略微增加，但已优化：
- 早报只分析港股+A股持仓（2-3只）
- 晚报只分析美股持仓（5只）
- 总增加约50-100 tokens/次
- 月度成本增加 <$0.05

### Q4: 可以通过钉钉直接发消息更新持仓吗？

**A:** 理论上可以，但需要：
1. 一个公网服务器
2. 部署Webhook接收器
3. 配置Outgoing机器人

**不推荐：** 直接用Excel或命令行更简单实用。
