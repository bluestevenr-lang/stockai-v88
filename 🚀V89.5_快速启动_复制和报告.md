## ⚡ 3分钟快速上手

### 步骤1: 启动应用

```bash
cd /Users/bluesteven/Desktop/StockAI
streamlit run app_v88_integrated.py
```

### 步骤2: 创建持仓模板（首次使用）

1. 打开应用后，侧边栏找到"💼 我的持仓"
2. 点击"📝 创建持仓模板"按钮
3. 等待提示"✅ 已创建持仓模板: my_portfolio.xlsx"
4. 打开`my_portfolio.xlsx`文件，编辑你的真实持仓
5. 刷新页面，查看持仓分析

### 步骤3: 使用一键复制功能

**场景A: 复制舆情分析**

1. 搜索股票（例如：`AAPL`）
2. 进入"⚔️ 深度作战室"
3. 找到"🏦 机构研究报告"
4. 切换到"📰 舆情分析" Tab
5. 点击"🚀 启动舆情分析"
6. 等待AI分析完成（10-20秒）
7. 在"📋 一键复制舆情报告"文本框中：
   - 全选：Ctrl+A (Windows) 或 Cmd+A (Mac)
   - 复制：Ctrl+C (Windows) 或 Cmd+C (Mac)
8. 粘贴到邮件/文档/笔记中

**场景B: 生成完整报告**

1. 在"深度作战室"中完成股票分析
   - 基本行情 ✅（自动）
   - 量化指标 ✅（自动）
   - 风险指标 ✅（自动）
   - Alpha因子 ✅（自动）
   - 机构研究 ⚠️（需手动启动）
   - 舆情分析 ⚠️（需手动启动）
2. 滚动到"📄 一键生成完整报告"部分
3. 点击"📋 生成完整报告"按钮
4. 查看报告预览（可展开/收起）
5. 在"📋 复制完整报告"文本框中：
   - 全选：Ctrl+A / Cmd+A
   - 复制：Ctrl+C / Cmd+C
6. 分享给团队/客户

- ✅ 基本行情（必有）
- ✅ 量化指标（必有）

## 📞 常见问题 FAQ

### Q1: 为什么报告中某些部分显示"N/A"？

**A**: 该部分的分析未启动。

**解决**:
1. 检查是否启动了机构研究分析
2. 检查是否启动了舆情分析

### Q2: 复制后粘贴，格式乱了？

**A**: 不同应用对Markdown的支持不同。

**解决**:
1. **推荐**: 粘贴到Markdown编辑器（Typora/Obsidian）

## 🔧 故障排查

### 问题1: 创建持仓模板失败

**症状**: 点击"创建持仓模板"无反应或报错

**检查**:
```bash
python3 -c "import openpyxl; print('✅ openpyxl已安装')"
```

**解决**:
```bash
pip3 install --trusted-host pypi.org --trusted-host files.pythonhosted.org openpyxl
```

---

### 问题2: 报告生成失败

**症状**: 点击"生成完整报告"后显示错误

**可能原因**:
1. 股票数据获取失败
2. 分析数据不完整

**解决**:
1. 检查网络连接
2. 检查代理设置（port: 1082）
3. 重新搜索股票
4. 重新分析

---

### 问题3: 复制后内容为空

**症状**: 复制后粘贴，内容是空白

**可能原因**:
1. 浏览器权限问题
2. 复制方法不对

**解决**:
1. 使用文本框复制方式
2. 先点击文本框
3. Ctrl+A 全选
4. Ctrl+C 复制
5. Ctrl+V 粘贴
